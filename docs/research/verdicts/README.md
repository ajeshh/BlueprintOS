# Research verdicts — what `/vet` decided

One `RVW-NNN-*.md` per vetted claim. Each records the claim, the rubric findings, and one honest
verdict: **ADOPT** (handed to `/boss-learn`), **ADAPT** (modified, reasoned), **REJECT** (with the
killing reason), or **NOT-YET** (with a re-open condition).

This folder is BOSS's institutional memory of what it **deliberately did not adopt**. Before vetting
a new claim, `/vet` reads here — so the same Reddit thread doesn't get re-litigated every month.

See [`/vet`](../../../.claude/skills/vet/SKILL.md) and
[IDEA-016](../../ideas/IDEA-016-research-intake-and-vetting.md).
